# Шаблоны Issue для GitHub

Ниже — 4 готовых шаблона. Перед публикацией разложите их по файлам в папке
`.github/ISSUE_TEMPLATE/` (точка в начале обязательна — иначе GitHub их не увидит):

```
.github/ISSUE_TEMPLATE/
├── new_character.md   ← «Новый персонаж»
├── new_connection.md  ← «Новая связь»
├── new_title.md       ← «Новый фильм / сериал / комикс»
└── bug_report.md      ← «Ошибка в данных или интерфейсе»
```

Каждый шаблон начинается со своего YAML-заголовка (`---`). Скопируйте блок целиком в соответствующий файл.

---

## `new_character.md`

```markdown
---
name: "➕ Новый персонаж"
about: Предложить добавить персонажа на карту
title: "[Персонаж] Имя"
labels: ["новый персонаж", "данные"]
---

**Имя (EN / RU):**
**Настоящее имя:**
**Актёр (если экранный):**
**Вселенная:** <!-- Earth-199999 (MCU) / Earth-616 / Earth-828 / Spider-Verse / Earth-10005 / другая -->
**Группа:** <!-- avengers, x_men, villain, cosmic, spider_verse, street_defenders... -->
**Ссылка на изображение (Fandom):** <!-- .../revision/latest?cb=... -->
**Ссылка на статью Fandom:**

**Связи** (по одной на строку — `тип — с_кем — подпись`; типы: team, family, romantic, ally, enemy, variant):
​```
team — star_lord — Стражи Галактики
variant — nova_616 — мультивселенная
​```
```
