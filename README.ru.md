## Что это

**Карта Мультивселенной Marvel** — одностраничное веб-приложение без зависимостей, раскладывающее всю вселенную Marvel в виде исследуемого графа. Оно отвечает на вопросы, которые фанаты задают на самом деле:

- **В каком порядке смотреть фильмы и сериалы?** → вид по внутренней хронологии.
- **В каком порядке читать комиксы?** → 166 выпусков в 12 линейках, по годам выхода.
- **Как связан персонаж A с персонажем B?** → граф связей с поиском кратчайшего пути.
- **Как пересекаются вселенные?** → MCU, Земля-616, Земля-828, Spider-Verse, Sony, Fox, What If…? и другие, с цветовой кодировкой.

## ✨ Возможности

- **Три режима** — Персонажи (граф связей), Истории (таймлайн фильмов/сериалов), Комиксы (каталог по линейкам или по годам).
- **Две раскладки персонажей** — силовой граф или «звезда» по вселенным с MCU в центре.
- **Поиск пути** 🧭 — кратчайшая цепочка связей между любыми двумя героями (например, Человек-паук → Тор).
- **Глобальный поиск** сразу по персонажам, актёрам, фильмам и комиксам.
- **Подробные карточки** — актёры, появления, принадлежность, кросс-ссылки между режимами, «где посмотреть / читать».
- **Двуязычность** — русский и английский (расширяемый выпадающий список).
- **Мобильный интерфейс** — нижнее меню, выезжающие панели, закрытие тапом вне окна.
- **Горячие клавиши** — `Esc` сброс · `/` поиск · `1` / `2` / `3` режимы · `R` сброс вида.
- **Прямые ссылки** — `…/#iron_man` сразу открывает персонажа.
- **Полностью статично** — без бэкенда и сборки. Просто открой файл.

## 🎬 Три режима

**Персонажи.** Силовая симуляция D3 из 360 героев и 710 связей: команда, семья, романтика, союзники, враги, альт-версии. Размер узла — число связей, цвет кольца — вселенная, заливка — фото.

**Истории.** 139 фильмов и сериалов на таймлайне — по фазам MCU или по внутренней хронологии, с полосами вселенных, где видно, как переплетаются линии Sony, Fox и анимации. Это и есть порядок просмотра.

**Комиксы.** 166 ключевых выпусков в 12 линейках (Человек-паук, Люди Икс, Железный человек, Тор, Капитан Америка, Мстители, Космос, Улицы, а также прологи к фильмам, адаптации, первые появления и классические события) — по линейкам или на общей оси лет.

## 🚀 Запуск

Это статический сайт — без установки и сборки.

```bash
open index.html             # macOS — просто открыть
# или локальный сервер:
python3 -m http.server 8000  # затем http://localhost:8000
```

Единый файл `Marvel_Universe_Map_v6.html` (всё встроено) открывается двойным кликом где угодно.

## 📁 Структура проекта

```
index.html    разметка, SEO-мета, структурированные данные, подключение стилей и скриптов
styles.css    все стили и тёмная тема
data.js       данные (window.DATA): персонажи, истории, комиксы, связи
app.js        вся логика — D3-граф, режимы, поиск, фильтры, карточки, мобильный UI
og-image.png  превью для соцсетей (1200×630)
robots.txt · sitemap.xml · .nojekyll   для поисковиков и GitHub Pages
```

Данные лежат в `data.js` одним объектом `DATA`:

```js
DATA.characters.nodes  // { id, name, name_ru, actor, group, universe, image, … }
DATA.characters.edges  // { source, target, type, label }   type: team|family|romantic|ally|enemy|variant
DATA.stories.nodes     // { id, title, title_ru, type, phase, date, poster, characters[] }
DATA.comics.nodes      // { id, title, title_ru, line, date, cover, tie_in, tie_in_chars[] }
```

### Добавить персонажа

1. Добавьте объект в `DATA.characters.nodes` в `data.js`.
2. Добавьте хотя бы одну связь в `DATA.characters.edges`.
3. Обновите страницу — граф пересчитается сам.

Не хотите трогать код? Откройте **Issue** по шаблонам из [`ISSUE_TEMPLATES.md`](ISSUE_TEMPLATES.md).
