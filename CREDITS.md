# Credits

**Проект:** Карта Мультивселенной Marvel — интерактивная визуализация связей между персонажами, фильмами, сериалами и комиксами Marvel.

**Зачем:** удобно исследовать, кто с кем связан, как пересекаются вселенные (MCU, Земля-616, Spider-Verse, Земля-828, Sony, Fox) и в каком порядке смотреть фильмы; наглядная «карта входа» в мультивселенную для новых зрителей и фанатов.

**Автор:** Кирилл — идея, сбор и выверка данных, дизайн.

**Технологии:** HTML + ванильный JavaScript в одном файле, визуализация графа на [D3.js v7](https://d3js.org) (force-simulation, zoom, drag), SVG-рендеринг, CSS-переменные для тёмной темы, localStorage для настроек.

**Данные и изображения:** [Marvel Cinematic Universe Wiki](https://marvelcinematicuniverse.fandom.com), [Marvel Database](https://marvel.fandom.com), [Into the Spider-Verse Wiki](https://intothespiderverse.fandom.com).
